Propiedades básicas de CSS

Objetivo general

Presentar la propuesta de diseño para el Objeto Virtual de Aprendizaje (OVA) sobre las propiedades básicas de CSS.

Objetivos específicos

  • Identificar las propiedades básicas de CSS a trabajar.
  • Determinar los recursos que se utilizarán para el desarrollo del Objeto Virtual de Aprendizaje.
  • Maquetar mediante la herramienta Moqups el diseño del OVA.

Materiales

Se utilizarán los siguientes recursos para ayudar a la comprensión de la temática seleccionada:

Lecturas:

  • Diseño web con CSS - Capítulo 3
  • Manual de CSS - Hojas de estilo
  • CSS propiedad display

Imágenes:

  • Flaticon
  • Freepik
  • Pixabay
  • Unsplash

Videos:

  • Uso de la propiedad display
  • Guía sobre display en CSS

Contenido

La página constará de un menú que contendrá las siguientes opciones:

Inicio:

En esta sección se dará la bienvenida a los usuarios y se les indicará la ruta a seguir para comenzar el proceso de aprendizaje de las propiedades básicas de CSS.

Lecturas:

Aquí se encontrará la explicación teórica de la temática, se enseñará acerca de las propiedades básicas de CSS como display, derivados de font, width, height, background, entre otros. Adicionalmente, se recomendarán algunas lecturas para el usuario que desee profundizar.

Recursos:

Se brindará al usuario contenido multimedia como imágenes y videos para que el aprendizaje sea más interactivo.

Ejemplos:

En esta sección se presentarán variados ejemplos gráficos acerca del uso de las distintas propiedades de CSS.

Evaluación:

En la sección evaluación el usuario podrá poner a prueba los conocimientos adquiridos, mediante un cuestionario que al finalizar, entregará al usuario su respectiva realimentación.

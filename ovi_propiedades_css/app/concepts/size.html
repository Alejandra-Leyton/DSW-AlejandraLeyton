Tamaño


Permite indicar el tamaño que tendrá un elemento.

El tamaño puede ser indicado en porcentaje (%), pixeles (px), entre otras medidas.



  • height


    • Designa la altura de un elemento.


    Por ejemplo: height: 100px.

    Esto quiere decir que al elemento que se le indique esta altura, ocupará un espacio de 100 pixeles de alto.



  • width


    • Designa el ancho de un elemento.


    Por ejemplo: width: 100%.

    Esto quiere decir que al elemento que se le indique este ancho, ocupará el 100% del espacio del lugar en el que se encuentre.

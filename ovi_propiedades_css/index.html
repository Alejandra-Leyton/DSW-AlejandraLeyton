¡Hola!, te doy la bienvenida una vez más al curso de diseño de sitios web.


El día de hoy te voy a enseñar las propiedades básicas de CSS y te voy a explicar cómo utilizar cada una, para que así puedas darle estilo a tus páginas web.


Comenzaremos por indicarte los objetivos de este proyecto:


Objetivo general

Enseñar las propiedades básicas de CSS.


Objetivos específicos

  • Identificar las propiedades básicas de CSS.

  • Explicar el concepto clave de cada propiedad.

  • Brindar recursos para profundizar en el aprendizaje.


Te invito a que continúes navegando por el sitio web para que conozcas más sobre las propiedades básicas de CSS. Te recomiendo comenzar con la propiedad Display.

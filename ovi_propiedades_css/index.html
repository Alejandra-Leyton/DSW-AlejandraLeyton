¡Hola!, te doy la bienvenida una vez más al curso de diseño de sitios web.


El día de hoy te voy a enseñar las propiedades básicas de CSS y te voy a explicar cómo utilizar cada una, para que así puedas darle estilo a tus páginas web.


Comenzaremos por indicarte los objetivos de este proyecto:


Objetivo general

Presentar la propuesta de diseño para el Objeto Virtual de Aprendizaje (OVA) sobre las propiedades básicas de CSS.


Objetivos específicos

  • Identificar las propiedades básicas de CSS a trabajar.

  • Determinar los recursos que se utilizarán para el desarrollo del Objeto Virtual de Aprendizaje.

  • Maquetar mediante la herramienta Moqups el diseño del OVA.
